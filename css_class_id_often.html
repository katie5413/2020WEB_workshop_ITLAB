class與id還有常用屬性篇

.Class / #Id

簡單來說 直接指定style > #id > .class > 普通標籤：

HTML

我是p

這裡的同階層的順序不影響結果，真正的戰場在css

我是class="blue ivory underline"的p

我是class="blue" id="red"的p

我是class="blue" id="red" style="color:green;"的p

CSS
p{ color:black; }
.ivory{ color:ivory;}
/*blue 比 ivory 還後面，所以會蓋掉前面的*/
.blue{ color:blue; }
/*雖然同階層，但結果不衝突，所以會1+1=2*/
.underline{ text-decoration:underline; }
#red{ color:red; }

常用屬性

文字類

color：文字顏色;
font-size：文字大小;
text-decoration：文字裝飾;//bold,underline,italic

其中，顏色又有許多表現方式

  • color：預設顏色; //red,green,#FFFFF.. 有興趣可以去看常用顏色表
  • color：rgb(red,green,blue); //rgb(255,255,255)＝white
  • color：rgba(red,green,blue,alpha); //rgb(0,0,0,0.3)
