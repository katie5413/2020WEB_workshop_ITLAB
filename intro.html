HTML簡介篇

HTML

HTML (HyperText Markup Language)直譯為「超文字標示語言」

超文字(HyperText)

可藉由一些<特殊標籤>的協助，可以讓使用者以網狀（Web）方式來編輯文件。

標示語言 (Markup Language)

透過在文章中插入標籤 (Tags) 來賦予文字特性，如標題、連結...等。標籤是由「小於」(<)跟「大於」(>)這兩個符號所括住的文字

HTML的語言形式為尖括號包圍的HTML元素（如 < html > ），瀏覽器使用HTML標籤和指令碼來詮釋網頁內容，但不會將它們顯示在頁面上。

網頁起手式：

<!doctype html>
<html>
<title> 我是標題 </title>
<head> 主要放外部連結，如：CSS/JS/Bootstrap等 </head>
<body> 大部分時間我們都在處理這裡要放的東西 </body>
</html>
